Simple Example Java Program

public class Manager{
    public static void main(String[] args) {
        System.out.println("WELCOME TO WEB DEVELOPMENT");
    }
}

Explanation Of Java Programm:

This is a simple example of Java program that prints "WELCOME TO WEB DEVELOPMENT" to the screen.

  • public class Manager: Defines the java class.
  • public static void main: This is the entry point of the program that defines main method.
  • System.out.println:This Prints the message to the screen.
